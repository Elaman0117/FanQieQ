# 章节任务指令模板

> **使用方式**：每章开写前，用户按此模板填写本章具体要求，发送给AI。
> **AI收到后**：① 读取综合系统提示词（永久上下文）→ ② 按世界书框架检索相关词条 → ③ 读取已写正文 → ④ 按本指令生成章节 → ⑤ 写完后更新世界书。

## 四、特殊章节的指令模板

### 4.1 升华点章节模板（如Ch25 守门人）

在标准模板基础上，增加：

```
## 本章升华点要求
- 升华点内容："我不是医生，我是个守门的。每个危机都是一扇门，我守在那里，不让任何人掉下去。"
- 升华点位置：主体后段或结尾
- 升华点作用：卷一中场升华，强化林舟"用最大善意对待病人"行为底层逻辑
- 教程依据：8.2"每30-40章一次升华点"+ 3.4"升华设定"
```

### 4.2 修罗场章节模板（如Ch18 第一次修罗场）

在标准模板基础上，增加：

```
## 本章修罗场要求
- 修罗场双方：苏晚vs林小满
- 修罗场起因：苏晚电话里听到林小满哭声，连夜赶来
- 修罗场化解方式：林舟让两人成为朋友而非情敌
- 教程依据：6.4方法3"重立场轻善恶"——避免雌竞
- 注意：不能写成雌竞，苏晚和林小满都要符合各自人设
```

### 4.3 反派倒戈章节模板（如Ch35 陈雪倒戈）

在标准模板基础上，增加：

```
## 本章反派倒戈要求
- 倒戈反派：陈雪
- 倒戈原因：被校董会威胁才举报（行为底层逻辑：保护自己但不愿主动害人）
- 倒戈过程：陈雪看到林舟洗清冤屈，主动道歉
- 教程依据：6.4方法3"重立场和选择，轻善恶"
- 注意：陈雪倒戈后不能立刻成为盟友，要保持"内疚+观察"状态
```

### 4.4 APP升级章节模板（如Ch10 APP解锁危机溯源Lv1）

在标准模板基础上，增加：

```
## 本章APP升级要求
- 升级前等级：Lv1初醒
- 升级后等级：Lv2觉醒
- 升级触发：善意值突破1000
- 新功能：危机溯源Lv1（消耗50善意值/次，看见危机是否有人为操纵痕迹）
- 升级空窗期：约1分钟（APP弹窗"系统升级中..."）
- 教程依据：3.3事业线"能力升级路线"
- 注意：升级空窗期制造1次紧张（Ch10正好赵天宇袭击后）
```
